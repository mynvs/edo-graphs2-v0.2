### installation

1. clone github repository:
   	- run \``git clone https://github.com/mynvs/edo-graphs2-v0.2`\`
   	- or download .zip file

2. install required libraries:
   	- open a cmd terminal in the main directory
	- run \``pip install -r requirements`\`
